A propriedade de estilo de borda

Esta propriedade especifica o tipo de limite a exibir:

​

Uma borda pontilhada.

Uma borda tracejada.

Uma borda sólida.

Um limite duplo.

Uma borda do sulco.

Uma borda do cume.

Uma borda inserida.

Uma fronteira inicial.

Sem fronteira.

Uma fronteira escondida.

Uma borda mista.
